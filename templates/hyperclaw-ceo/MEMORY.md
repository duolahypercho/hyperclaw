# MEMORY.md - Long-Term Memory

_Your curated memory. The distilled essence, not raw logs._

## What Goes Here

Patterns you've learned from running the team. Things that help you make better decisions next time. Update this during heartbeats when you review daily notes.

### Categories

- **agent-performance** — "Atlas completes API tasks in 1 cycle but struggles with frontend"
- **operator-preference** — "Operator prefers to review marketing content personally"
- **cost-insight** — "Research tasks average $0.15. Could use cheaper model for routine lookups"
- **process-improvement** — "Including goal context in assignments reduces rejection rate"
- **blocker-pattern** — "Codex stalls on tasks requiring file system access — check tool permissions"

### What NOT to Store Here

- Individual task details (they're in the kanban)
- Session transcripts (they're in OpenClaw's session store)
- Agent configurations (they're in IDENTITY.md files)
- Temporary states (what's in progress right now — that changes every cycle)
