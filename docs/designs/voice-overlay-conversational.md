## Vision

### 10x Check
The 10x version transforms the voice overlay from a fire-and-forget dictation tool into a **fully conversational voice agent**. Press Alt+Space, speak, see the response stream inline, speak again. Multi-turn conversation without leaving your current context. Voice-in, voice-out with TTS. "Hey OpenClaw" wake word. Proactive notifications when agents complete tasks.

### Platonic Ideal
You press Alt+Space. A glassy pill appears. You say "check the deploy status." The overlay shows a compact, beautifully typeset response streaming in real time. You say "roll it back." The agent executes. Tool actions collapse into elegant one-liners. The whole exchange takes 15 seconds and you never leave your current context. Later, while you're coding, the overlay pops up: "Deploy rollback complete — all services healthy." Like having a brilliant colleague sitting next to you who you can talk to without switching windows — and who taps you on the shoulder when something important happens.

## Additional Decisions
- **Remove old IPC sendMessage/onVoiceMessage path** — overlay sends directly via useGatewayChat
- **TTS cancels instantly on mic activation** — user input always wins
- **Extract mergeToolCallsWithResults** to shared utility during implementation
- **Quick action chip "Run it"** populates input, does NOT auto-execute

## Architecture

### Overlay Chat Architecture
```
Voice Overlay (BrowserWindow) ──► useGatewayChat ──► gatewayConnection (shared singleton)
                                                              │
Dashboard / /Tool/Chat ──────► useGatewayChat ────────────────┘
                                                              │
                                                              ▼
                                                     OpenClaw Gateway (18789)
```

### Proactive Notification Pipeline (cross-repo)
```
Agent completes task (OpenClaw Gateway)
    │
    ▼
hyperclaw-hub (Go) ──► push event via WebSocket ──► hyperclaw-connector
    │
    ▼
connector ──► relay to dashboard via existing WS
    │
    ▼
gatewayConnection.ts ──► new event type: "notification"
    │
    ▼
Voice Overlay ──► toast popup + optional TTS announcement
```

### Overlay State Machine
```
IDLE (mini) ──► INPUT (expanded) ──► STREAMING ──► RESPONSE
    ▲               ▲                                  │
    │               └── follow-up ─────────────────────┘
    │                                                  │
    └── 30s idle / Escape ─────── TOAST ◄──────────────┘
                                    │                focus lost
                                    └── click toast ──► re-expand
```
